Open the `Extensions` sidebar in VS Code (or Ctrl+Shift+X). <br>
Search for `Floptropica Color Theme`, choose `"Floptropica Color Theme"` by `coramen(me)`.<br>
Click `Install` to installтЬи.<br>
Navigate to `Settings` -> `Themes` -> `Color Theme `-> `"Floptropica Color Theme"`.<br>
